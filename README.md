# Student Management System

## 📖 Description
This is a simple Student Management System built using Python. It helps users manage student records.

## ✨ Features
- Add Student
- View Students
- Search Student
- Delete Student
- Exit Program

## 💻 Technologies Used
- Python

## 🚀 How to Run
1. Run the student_management.py file.
2. Choose an option from the menu.
3. Enter student details.
4. View, search, or delete student records.
